## 核心逻辑

```
账号被删 / Cookie 过期
  → 查找该账号的所有活跃池链接 (is_pool=1, status='active')
  → 排除当前账号，按负载最低选替换账号
  → 逐个更新 share_links.account_id
  → 返回切换结果给前端展示
```
