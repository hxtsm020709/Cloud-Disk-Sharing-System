# 网盘共享系统 — UI 设计规范

> 版本：v1.0  
> 更新日期：2026-06-04

---

## 1. 设计原则

- **简洁**：每个页面只做一件事，减少视觉噪音
- **直观**：按钮和操作符合用户直觉，不需要学习成本
- **响应式**：管理端适配桌面（>=1024px），扫码端适配手机（375-414px）

---

## 2. 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#4A90D9` | 淡蓝色，用于导航栏、主按钮、强调元素 |
| 主色浅 | `#E8F1FB` | 淡蓝背景，用于卡片、区块底色 |
| 主色深 | `#357ABD` | 深蓝，用于按钮 hover、链接 |
| 成功 | `#52C41A` | 绿色，Cookie 有效、链接正常 |
| 警告 | `#FAAD14` | 橙色，即将到期、待处理 |
| 危险 | `#FF4D4F` | 红色，Cookie 过期、链接失效 |
| 中性黑 | `#333333` | 主要文字 |
| 中性灰 | `#999999` | 次要文字、占位符 |
| 边框 | `#E8E8E8` | 分割线、输入框边框 |
| 背景 | `#F5F7FA` | 页面底色 |

## 3. 页面布局

### 3.1 管理端通用布局

```
┌─────────────────────────────────────────┐
│  导航栏（淡蓝 #4A90D9，白色文字）       │
│  [Logo]  账号管理 | 分享链接 | 退出     │
├─────────────────────────────────────────┤
│                                         │
│        页面内容区（白色卡片）            │
│                                         │
│                                         │
├─────────────────────────────────────────┤
│  页脚 © 2026                          │
└─────────────────────────────────────────┘
```

- 导航栏固定在顶部
- 内容区域使用白色卡片 + 圆角 + 微阴影
- 侧边无菜单，顶部导航即可（页面数量少）

### 3.2 手机扫码端布局

```
┌───────────────────┐
│   状态栏           │
├───────────────────┤
│                   │
│   ┌─────────────┐ │
│   │  摄像头预览  │ │
│   │  (取景框)    │ │
│   │             │ │
│   │   [扫描线]  │ │
│   │             │ │
│   └─────────────┘ │
│                   │
│  请将二维码对准    │
│  取景框            │
│                   │
│  [状态提示文字]    │
│                   │
└───────────────────┘
```

- 全屏高度，摄像头预览区域占主要面积
- 扫描线动画提示正在工作
- 底部状态提示
- 白色/淡蓝背景

## 4. 组件规范

### 4.1 按钮

| 类型 | 样式 | 用途 |
|------|------|------|
| 主按钮 | 淡蓝背景、白色文字、圆角 6px | 主要操作（保存、生成链接） |
| 次按钮 | 白色背景、淡蓝边框、淡蓝文字 | 次要操作（取消、返回） |
| 危险按钮 | 红色背景、白色文字 | 删除、停用 |
| 小按钮 | 同主按钮，padding 缩小 | 表格内操作 |

### 4.2 表单

- 输入框：白色背景、`#E8E8E8` 边框、圆角 6px、focus 时边框变淡蓝
- 标签：置于输入框上方、字号 14px、颜色 `#666`
- 错误提示：红色文字，位于输入框下方

### 4.3 卡片

- 白色背景、圆角 8px、阴影 `0 2px 8px rgba(0,0,0,0.08)`
- 内边距 20px
- 标题字号 18px bold

### 4.4 状态标签

- Cookie 有效：绿底绿字
- Cookie 过期：红底红字
- 链接有效：绿底绿字
- 链接过期：灰底灰字
- 会员即将到期：橙底橙字

### 4.5 表格

- 表头：`#F5F7FA` 背景、加粗
- 行 hover：`#E8F1FB` 背景
- 分割线：`#E8E8E8`
